學校發展簡史

民國53年，總統府前國策顧問吳三連先生鑒於國家經濟首要之務乃在於人才培植，遂邀請台南地方名士：辛文炳、侯雨利、吳修齊、吳尊賢、吳俊傑、侯永都、陳清曉、莊昇如、陳旗安、張麗堂等政治、經濟、文化知名人士發起籌設，購地(五甲餘)於台南市永康區六甲頂。民國58年，正式設立二年制工業技藝專科學校。民國85年改制為「南臺技術學院」，旋於民國88年改名為「南臺科技大學」，民國106年8月21日起改名為南臺學校財團法人南臺科技大學。

建校群賢

  • 吳三連 先生

    首任董事長
    • 吳三連 先生
    • 首任董事長任期：1969～1989年


    • 吳三連先生，民國前12年出生於台南學甲（今台南市學甲區），東京商科大學（今一橋大學）畢業。為前台灣省議員、台北市長及自立晚報、台南紡織創辦人，亦為日治時期與戰後台灣民族運動、社會運動及政治運動的先驅人物。先生畢生力行辦教育就是要犧牲付出，辦學校就要辦一所理想的學校，絕不可誤人子弟，更不可以校斂財的信念。

    • Read More

  • 辛文炳 先生

    第二任董事長
    • 辛文炳 先生
    • 第二任董事長任期:1989～1999年


    • 辛文炳先生，民國元年2月1日誕生在台南市開山路馬公廟附近。自幼聰慧過人，稟性溫厚。九歲就讀台南第三公學校（今進學國小），後考取台南州立第二中學（今台南一中）；十九歲時負笈東瀛，民國25年畢業於明治大學法學部。民國78年先生出任南臺科技大學董事，即時時關心教育工作的改革與成效。

    • Read More

  • 高清愿 先生

    第三任董事長
    • 高清愿 先生
    • 任期：1999～2014年


    • 高清愿先生，民國18年5月24日出生於台南縣學甲鎮。自民國61年起即出任本校董事，關心教育工作。強調教育必須企業化，要以市場導向的觀念改革教育，使之國際化、自由化、市場化及社會化。民國88年4月16日，本校董事會倚重其人望與卓越領導能力，推舉先生擔任本校第三任董事長。

    • Read More

  • 鄭高輝 先生

    第四任董事長
    • 鄭高輝 先生
    • 任期：2014～2018年


    • 鄭高輝先生，民國22年生於臺南市。年少家境不豐，身為長男，為減輕父母經濟負擔，選擇就讀職業學校，俾便習得專業知識，提早進入職場。民國60年投資臺南紡織公司，任董事兼副總經理，民國64年升任臺南紡織常務董事兼總經理，民國78年起晉升為副董事長兼總經理，民國82年從總經理職務屆齡退休，擔任副董事長兼南紡集團總裁，民國93年獲聘董事長。

    • Read More

  • 辛文炳 先生

    首任校長
    • 辛文炳 先生
    • 首任校長任期：1969年5月～1989年4月


    • 辛文炳先生，民國元年2月1日誕生在台南市開山路馬公廟附近。民國五十四年，董事長吳三連先生邀請南部工商鉅子和社會名流吳修齊等十五人，共同捐資興學，於台南市郊六甲頂購地建校，但因政府法規限制，直至民國58年11月才奉准立案招生，創校初期，學校在財政極度困難的窘境，但先生在董事會的託付下，憑著堅忍不拔的毅力，配合全校師生的拓荒精神，篳路藍縷，以啟山林。

  • 張信雄 博士

    第二任校長
    • 張信雄 博士
    • 第三任校長任期：2007年8月～2017年11月


    • 張信雄博士，台南縣人，民國二十九年三月三日生，國立政治大學國際貿易系學士、企業管理碩士，日本東京農業大學農業經濟博士。張信雄先生於民國75年(1986年)獲聘為南臺工專副校長，任內除了對提升學校建設及校園美化多所貢獻外，更首開專科學校引進博士級師資之先例，奠定日後本校師資完備之基礎。三年後(1989年)更因積極辦學及校務經營之能力深受本校董事會肯定，擢升校長。

    • Read More

  • 戴謙 博士

    第三任校長
    • 戴謙 博士
    • 第三任校長任期：2007年8月～2017年11月


    • 戴謙博士，高雄縣人，民國38年8月20日生。民國96年，南臺科技大學董事會遴選第三任校長，由於戴博士擔任國科會副主任委員任內策劃及推動國家科學研究發展業務，具有高科技產業豐富的經驗與人脈，受董事會遴聘為第三任校長，打造南臺科技大學成為第一流學府。

  • 盧燈茂 博士

    第四任校長
    • 盧燈茂 博士
    • 第二任董事長任期：1989～1999年


    • 盧燈茂博士，出生於嘉義新港鄉農家。盧燈茂校長為國立成功大學機械工程博士，民國74年至南臺科大任教，自擔任機械工程系講師起，一路由副教授升為教授，擔任機械工程系主任、機電科技研究所所長、奈米科技中心主任、工學院院長等學術主管，隨後擔任學術副校長、行政副校長等重要行政職務，對於校內事務嫻熟，對教育環境更是關注以及了解。
